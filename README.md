# learning
scarecrow.alpha: написанная мной для школьного проекта программа, суть которй заключается в том, чтобы использовать функции вирусов против них. в данной версии реализован способ защиты устройства, путём подмены параметров и создания ложных целей на которые будут реагировать вирусы при исследовании окружения и удаляться или останавливать работу.
в будущем эта программа будкет улучшаться.
